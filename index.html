Our Solution: AI Agent + RAG

A Smarter Approach

Our project uses AI to make the admission process easier by:

  • ‚úÖConnecting students with clear, instant answers.
  • ‚úÖHelping the university support them more efficiently.

How Our Agent Thinks

Key Improvements over Traditional RAG:

Dynamic Flow Control: Only runs retrieval when needed, avoiding unnecessary steps.

Smarter Context Selection: Selects from multiple databases to retrieve more relevant info.

Modular Agent Reasoning: The agent decides the next step instead of rigid linear execution.
